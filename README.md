# linear-regression

## How to Run

1. **Create a virtual environment**  
   ```sh
   python3 -m venv venv
   source venv/bin/activate
   ```

2. **Install dependencies**  
   ```sh
   pip install -r requirement.txt
   ```

3. **Train the model**  
   ```sh
   python train.py
   ```

4. **Estimate price**
    ```sh
   python estimate.py
   ```
